Landing page de um app de controle de estoque produzido em disciplina do curso de Análise e Desenvolvimento de Sistemas.

Um projeto no Unity em estágios iniciais de desenvolvimento. Embora atualmente eu não tenha planos de finalizar o projeto, ele foi esssencial para eu amadurecer como programador. Não utilizei nenhum asset, o que foi um grande desafio.

Modelo 3D minimalista low-poly produzido para o meu projeto pessoal.

Tarefa realizada para a disciplina de Desenvolvimento Web no curso de Análise e desenvolvimento de Sistemas. Basicamente o popular exercício de desenvolvimento web de clonar a página inicial do Netflix. Foi uma maneira interessante de exercitar a implementação de uma página web responsiva, preparada também para mobile.

Solução desenvolvida em meu estágio no Departamento Regional de Saúde XVI Sorocaba.

NotesJS

Web app de notas, produzido como um exercício para fazer um app sem o uso de nenhuma framework. Fiz uma versão que funciona localmente, salvando as notas nos cookies do navegador, e outra com o uso do Node.js, que salvaria as notas em um servidor. Abaixo, disponibilizo a versão que salva as notas localmente.
